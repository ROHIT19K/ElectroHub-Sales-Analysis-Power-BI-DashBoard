
# 📊  ElectroHub Sales Analysis 
________________________________________
## 📌 Project Overview

The ElectroHub Sales Data Analysis project focuses on analyzing retail sales data to uncover business insights related to sales performance, profitability, customer behavior, product trends, and regional performance.

This project was developed using Power BI to create an interactive and dynamic business intelligence dashboard that helps stakeholders monitor KPIs, compare performance metrics, identify high-performing products, and make data-driven business decisions.
The dashboard provides detailed analysis across multiple product categories such as Electronics, Footwear, Clothing, Home Appliances, Accessories, Kitchenware, Bags, and Personal Care.


## 🎯 Business Problem

Businesses often struggle to track sales trends, profit, Sales Over Time, and Discount On product, Sales By Different Cities .

This project solves the problem by:

•	Top/Bottom 5 Product By Sales/Profit/Quantity Sold

•	How Do Sales Trends Vary Over Time

•	Show Relationship between Sales and Profit

•	Compare sales/profit/quantity sold between any two periods selected by the user

•	Average discount offered in each discount category.

•	Total number of orders.

•   Show Sales/Profit/Discount/Net Sales/All remaining fields for each order that could be filtered using visual filters. (Product/Date/Customer ID/Promotion Categories)

•   Show sales by different cities.



## 📌 Project Objective

####  The main objective of this project is to:
    
•	Analyze sales/Profit/Quantity Sold performance By Top/Bottom 5 Product

## 🛠 Tools & Technologies

• 📊 Power BI Desktop – Main data visualization platform used for report creation.

• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.

• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.

• 📝 Data Modeling – Relationships established among tables (dim_customers, dim_products, dim_promotion, fact_sales) to enable cross-filtering and aggregation.

## 📂 Dataset Information

📌 Dataset Source
•	Store Sales Dataset (Excel File)

📌 Dataset Categories
The Excel File Contains Multiple Files Such as 

    1. dim_customers (Customer ID, Customer Name, City, Sate, Pincode, EmailID, Phone Number)
    
    2. dim_promotion (PromotionID, Promotion Name, AdType, Coupon Code, Price Reduction Type)
    
    3. dim_products (ProductID, ProductName, Product Line, Price Per Unit(INR))
    
    4. fact_sales (Order ID, Date, CustomerID, PromotionID, ProductID, Unit Sold, Price Per unit, Total Sales, Discount Percentage, Discount, Net Sales, Profit, )
